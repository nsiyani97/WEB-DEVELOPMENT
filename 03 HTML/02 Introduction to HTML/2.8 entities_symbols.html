Reserved characters in HTML must be replaced with entities

  • white space: 
  • less than:<
  • greater than:>
  • ampersand&
  • euro€
  • copyright©
  • trademark®
  • double quotation mark"
  • single quotation mark'
  • pound£

    • Symbols or letters that are not present on your keyboard can be added to HTML using entities

  • ™ TRADEMARK
  • ← LEFT ARROW
  • ↑ UP ARROW
  • → RIGHT ARROW
  • ↓ DOWN ARROW
  • ♥ HEART
  • ♦ DIAMOND
  • ∏ N-ary product
  • ∅ Empty sets
  • ∃ There exists
  • 😀
  • 😂
