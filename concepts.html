30 Concepts â€” Details & Examples

1. Variables & Data Types

Variables store values. Use let and const. JavaScript data types: Number, String, Boolean, Null, Undefined, Object, Array, Symbol, BigInt.

Example 
// declare variables
let name = "Alice";
const age = 25;
let isStudent = true;
console.log(name, age, isStudent);

2. Operators

Arithmetic, comparison, logical and assignment operators help to compute and compare values.

Example 
let x = 10;
let y = 3;
console.log(x + y, x - y, x * y, x / y);
console.log(x > y, x === 10);
console.log(x % y);

3. Functions

Reusable blocks of code. Function declarations, expressions, and arrow functions.

Example 
function add(a, b) { return a + b; }
const mul = (a, b) => a * b;
console.log(add(2,3), mul(2,3));

4. Objects

Key-value pairs. Properties store values; methods are functions inside objects.

Example 
const person = {
  name: 'Alice',
  age: 25,
  greet() { console.log('Hi ' + this.name); }
};
person.greet();

5. Arrays

Ordered lists of values. Use array methods to manipulate them.

Example 
let nums = [1,2,3];
nums.push(4);
console.log(nums.map(n => n * 2));

6. Loops & Iteration

Use loops to repeat actions: for, while, for...of, for...in.

Example 
for (let i = 0; i < 3; i++) console.log(i);
for (let v of ['a','b']) console.log(v);

7. DOM Manipulation

Change HTML from JavaScript with document.querySelector, innerText, createElement, and events.

Example 
// run inside a browser console or file with HTML
const btn = document.createElement('button');
btn.innerText = 'Click me';
document.body.appendChild(btn);
btn.addEventListener('click', () => alert('Clicked!'));

8. Scope & Closures

Scope = where variables are visible. Closures let inner functions remember outer variables.

Example 
function counter() {
  let count = 0;
  return function() { count++; return count; };
}
const c = counter();
console.log(c(), c()); // 1, 2

9. ES6+ Features

Modern JS: let/const, arrow functions, template literals, destructuring, modules, classes.

Example 
const [a, b] = [1, 2];
const greet = name => `Hello ${name}`;
console.log(greet('Alice'));

10. Asynchronous JavaScript

Handling delayed operations with callbacks, Promises, and async/await.

Example 
async function example() {
  const res = await fetch('https://jsonplaceholder.typicode.com/todos/1');
  const data = await res.json();
  console.log(data);
}
// example(); // run in browser or environment with fetch

11. Events & Event Handling

React to user actions with event listeners. Use event delegation for multiple elements.

Example 
document.body.addEventListener('click', function(e) {
  if (e.target.matches('.item')) console.log('Item clicked');
});

12. Error Handling

Use try/catch to handle runtime errors and avoid crashes.

Example 
try {
  JSON.parse('bad');
} catch (err) {
  console.error('Parsing failed', err.message);
} finally {
  console.log('Cleanup if needed');
}

13. Prototypes & Inheritance

Objects inherit from other objects via prototypes. Functions have a prototype property used when creating objects with new.

Example 
function Person(name) { this.name = name; }
Person.prototype.greet = function() { console.log('Hi ' + this.name); };
const p = new Person('Alice');
p.greet();

14. this Keyword

Value of this depends on call site. Arrow functions do not bind their own this.

Example 
const obj = { name: 'A', f() { console.log(this.name); } };
obj.f(); // 'A'
const f = obj.f;
f(); // undefined (or global) in non-strict mode

15. Classes & OOP

Syntactic sugar over prototypesâ€”use class, constructor, extends.

Example 
class Animal { constructor(name){ this.name = name } speak(){ console.log(this.name + ' makes a sound')} }
class Dog extends Animal { speak(){ console.log(this.name + ' barks') } }
new Dog('Rex').speak();

16. Modules (import / export)

Split code into files. Use export and import in modern JS tooling or native modules in browsers.

Example 
// math.js
export function add(a,b){return a+b}

// main.js
import { add } from './math.js';
console.log(add(2,3));

17. Storage & APIs

Use LocalStorage / SessionStorage for small client-side data. Fetch APIs to communicate with servers.

Example 
localStorage.setItem('theme','dark');
console.log(localStorage.getItem('theme'));

// fetch example
// fetch('/api/data').then(r => r.json()).then(data => console.log(data));

18. JSON

JavaScript Object Notation â€” data format used by APIs. Use JSON.stringify and JSON.parse.

Example 
const obj = { name: 'Alice' };
const json = JSON.stringify(obj);
const back = JSON.parse(json);
console.log(back.name);

19. Event Loop & Execution

JS uses an event loop: call stack, task queue, microtask queue. Understand setTimeout, Promises and async timing.

Example 
console.log('start');
setTimeout(() => console.log('timeout'), 0);
Promise.resolve().then(() => console.log('promise'));
console.log('end');
// Output: start, end, promise, timeout

20. Advanced DOM

Traverse nodes, manage fragments, templates, performance techniques (delegation, virtual DOM in frameworks).

Example 
const frag = document.createDocumentFragment();
for(let i=0;i<100;i++){ const li=document.createElement('li'); li.textContent = 'Item '+i; frag.appendChild(li); }
document.getElementById('list').appendChild(frag);

21. Higher-Order Functions

Functions that accept or return other functions. Core to functional style: map/filter/reduce.

Example 
const nums=[1,2,3,4];
const evens = nums.filter(n => n%2===0);
console.log(evens);

22. Callback Functions

Pass functions as arguments for later execution (older async style).

Example 
function greet(name, cb){
  console.log('Hello '+name);
  cb();
}
greet('Alice', () => console.log('Callback ran'));

23. Hoisting

Declarations are processed before code runs â€” function declarations are hoisted; var is hoisted (but not initialized); let/const are block-scoped and not accessible before declaration.

Example 
sayHi();
function sayHi(){ console.log('Hi') }
// var x; is hoisted, but not its value
// console.log(x); // undefined if x declared with var below

24. new, call, apply, bind

Control this and create objects. bind creates a new function with bound this.

Example 
function say(prefix){ console.log(prefix + ' ' + this.name); }
const obj = { name: 'Alice' };
say.call(obj, 'Hi');
const bound = say.bind(obj, 'Hey');
bound();

25. Destructuring & Spread/Rest

Extract values easily; expand arrays/objects and gather rest parameters.

Example 
const [a, ...rest] = [1,2,3];
const obj = {x:1,y:2};
const copy = {...obj, z:3};
console.log(a, rest, copy);

26. Template Literals

Backtick strings with embedded expressions and multi-line support.

Example 
const name='Alice';
console.log(`Hello, ${name}!\nWelcome.`);

27. Iterators & Generators

Create custom iteration behavior and lazy sequences using function* and yield.

Example 
function* gen(){ yield 1; yield 2; }
for (const v of gen()) console.log(v);

28. Symbol & BigInt

New primitive types: Symbol for unique keys; BigInt for very large integers.

Example 
const id = Symbol('id');
const big = 123456789012345678901234567890n;
console.log(typeof id, typeof big);

29. Memory Management & Garbage Collection

JS automatically frees unreachable objects. Understand references and avoid memory leaks (e.g., large DOM references, global caches).

Example (advice) 
// Break references when done:
const el = document.getElementById('heavy');
// use it...
// then clear reference
// el = null; // free up (if variable was let)

30. Design Patterns

Common architecture patterns: Module, Factory, Singleton, Observer. Useful for organizing large applications.

Example â€” Module Pattern 
const Counter = (function(){
  let count = 0;
  return { inc(){ count++; }, get(){ return count; } };
})();
Counter.inc(); console.log(Counter.get());
